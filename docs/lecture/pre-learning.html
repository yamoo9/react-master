# 사전 학습 가이드

잠깐! React 프레임워크 학습에 앞서 "Front-End 마스터 코스" (opens new window)에서 다루는 웹 표준 기술과 개발 방법을 알고 있는지 검토해보세요.
웹 표준 기술 및 개발 방법을 잘 모른다면? React 공부에 앞서 Front-End 개발 학습이 필요합니다.

  • 웹 표준(Standards) 및 접근성(A11Y) 이해
  • 구조 디자인 (HTML5 마크업)
  • 표현 디자인 (CSS3 스타일링 + 레이아웃)
  • 반응형 디자인 (RWD)
  • 인터랙션 디자인 (JavaScript + DOM API)
  • 비동기 프로그래밍(AJAX) 및 보안 이슈(SOP/CORS), 우회 방법(JSONP)
  • RESTful 웹 서비스 (CRUD + REST API)
  • 모던 자바스크립트 (ES 6+)
  • Git 버전 관리 & GitHub 서비스

자세한 내용은 아래 강의 커리큘럼 항목을 살펴보세요.
